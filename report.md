# Report: GenAI Workflow for Drafting Customer Support Responses

## Project Goal

The goal of this project is to demonstrate a simple GenAI workflow for drafting customer support responses.

## Workflow Steps

1. Input: receive a customer support message
2. Classification: identify the main issue
3. Tone detection: estimate the customer's tone
4. Response drafting: create a polite response
5. Review: check whether the response is polite, clear, and professional

## Why This Workflow Matters

Customer support teams often receive many similar requests. A GenAI workflow can help create fast first-draft responses, which saves time and improves consistency.

## Beginner-Friendly Design

This project is intentionally simple:

- it uses only basic Python
- it does not require external libraries
- it explains the workflow in clear steps
- it is easy to modify for homework or classroom discussion

## Limitations

- The script does not use a real language model
- The classification is based on keywords only
- The drafted responses are templates, not true AI generations

## Possible Future Improvement

A future version could connect to an actual large language model API to generate more natural and flexible responses.
